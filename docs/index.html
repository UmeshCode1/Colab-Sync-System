‚ùì FAQ

Do I need to install anything?

No! Everything runs in the cloud. Just open a link and start working.

Is my data secure?

Yes. Credentials are stored in memory only and cleared when the session ends. Nothing is saved in notebook files.

Can I work offline?

The cloud-first approach requires internet. However, you can clone the repo locally and work with regular git commands.

How do I get a GitHub token?

Go to GitHub Settings ‚Üí Personal Access Tokens, create a token with "repo" scope, and copy it.

Can I use this for private repositories?

Yes! Just use a token with appropriate permissions for private repos.
